Программа для даления вотермарки из видео или фото при помощи MI-GAN. Поиск вотермарки осуществляется при помощи модели yolo или ручного ввода пользователя.

Способы поиска вотермарки на видео:
Ручной ввод пользователя. Пользователь сам выбирает область для маскирования вотермарки ввода координаты левого верхнего и правого нижнего угла. Пропускает процесс поиска вотермарки при помощи модели yolo. Подходит для вотермарок, статичных на протяжении всего видео. Очень быстро.
Поиск вотермарки в каждой сцене по средством анализа первого кадра сцены. Модель обнаруживает расположение вотермарки на первом кадре и использует эту информацию для всей сцены, на следующей сцене проверка осуществляется снова. Таким образом кол-во кадров необходимых для обработки при помощи модели yolo сокращается до кол-ва сцен. Подходит если вотермарка меняет свое расположение в каждой сцене. Реализовано через yolo + pyscenedetect(pyav). Быстро
Поиск вотермарки с пропуском кадров. Модель yolo ищет вотермарку в кадре, после чего ищет вотермарку через кадр, через 2, через 3, 5 и так далее пока вотермарка не сменит свое расположение (нужна защита от шума, незначительных изменений положения bbox). Когда вотермарка сменила расположение модель возвращается к тому моменту, на котором вотермарка находилась на прежнем месте и пытаестя найти момент смены положения вотермарки в кадре. При этом пропущенные кадры не остаются без обработки, для них используется вотермарка созданная на предыдущем "ключеовм" кадре. Этот способ позволяет пропускать кадры если вотермарка меняет свое расположение раз в несколько секунд. Ускоряет обработку видео. Медлено
Поиск вотермарки на каждом кадре. Необхоимо если вотермарка меняет свое расположение каждый кадр. Очень медлено.

Входные данные: изображение, видео, директория с изображениями

Фичи:
Обработка сразу нескольких вотермарок в кадре. Для каждой вотермарки создается свой bbox, маска. Обработка каждой маски отдельно или пакетно, наложение каждой маскировки на оригинальный кадр или на прозрачный фон (png sequence).
Пользователь вводит временные границы в которых надо обрабатывать видео.
Пользователь может получить пример коробки yolo или обработанного кадра из видео если введет --yolo_preview TIME или --migan_preview TIME
